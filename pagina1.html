HTML para Iniciantes

Discrição

O Curso HTML para iniciantes tem como objetivo fazer uma introdução ao HTML e aos principais conceitos relacionados à linguagem de marcação. Com este curso, você vai aprender a criar páginas web, utilizando e editando códigos em HTML. Aprenda o conteúdo básico sobre HTML de uma forma fácil e rápida, com todas as aulas seguindo um passo a passo.

O que você aprenderá

  • Criar documentos HTML
  • Conheça as tags de uma estrutura básica do HTML
  • Inserir e utilizar hyperlinks.
  • Crar tabelas com as tags TABLE, TR, TD
  • Formatar texto em páginas utilizando HTML.
  • Inserir imagens.
  • Criar listar ordenadas e não ordenadas.

    • Este curso inclui:

  • Vídeo sob demanda de 5,5 horas
  • Acesso total vitalício
  • Acesso no dispositivo móvel e na TV
  • Certificado de Conclusão
